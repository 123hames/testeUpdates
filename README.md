Como instalar:

Para instalar basta apenas transferir o ficheiro "passarSerie.jar" disponivel no repositorio, apos transferido basta abrir a aplicação normalmente que ira abrir corretamente e criar os ficheiros de armazenamento de dados necessarios.
Como pre requisito necessita de ter o java instalado no computador!
